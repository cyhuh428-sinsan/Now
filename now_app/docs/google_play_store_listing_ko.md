# NowNote Google Play 등록 문구 초안

## 앱 이름
NowNote

## 짧은 설명
오늘의 메모, 음성 기록, 일정과 할 일을 한곳에 정리하는 개인 기록 앱

## 전체 설명
NowNote는 빠르게 기록하고, 나중에 다시 찾기 쉽게 정리하기 위한 개인 기록 앱입니다.

날짜별 간단 메모, 계층형 지식 메모, 음성 기록, 일정과 할 일을 한 화면에서 관리할 수 있습니다. 일상에서 떠오른 생각, 회의나 대화의 핵심, 여행과 살림 기록처럼 흩어지기 쉬운 정보를 NowNote에 모아두세요.

주요 기능
- 날짜별 메모: 하루 단위로 메모를 계속 추가하고 다시 확인
- 계층 메모: 부모/자식/손자 구조로 지식과 아이디어 정리
- 음성 기록: 말로 남긴 내용을 텍스트로 변환하여 저장
- 일정과 할 일: 오늘 일정, 루틴, 할 일을 홈에서 확인
- LLM 브리핑: 사용자가 기록한 정보를 기반으로 오늘의 참고 사항 제공
- 로컬 중심 사용: 기본 기록은 기기 안에서 관리
- 선택형 서버 연결: 사용자가 설정한 NowNote 서버와 메모, 녹음, 분석 작업을 동기화

NowNote는 입력을 줄이고 기록을 쉽게 만드는 것을 목표로 합니다. 복잡한 지식 관리 도구보다 가볍고, 단순한 메모장보다 연결된 개인 기록 환경을 제공합니다.

## 검색 키워드 참고
메모, 음성 메모, 일정 관리, 할 일, 지식 관리, 기록, 노트, 루틴, 브리핑

## 스크린샷 추천
- 홈 화면: 오늘 브리핑, 오늘 할 일, 오늘 일정, 오늘 메모
- 메모 탭: 달력 기반 일자별 메모
- 계층 메모: 부모/자식/손자 구조
- 음성 메모 입력 화면
- 설정 또는 권한 안내 화면
