### 전체 설명
```text
NowNote는 빠르게 기록하고, 나중에 다시 찾기 쉽게 정리하기 위한 개인 기록 앱입니다.

날짜별 간단 메모, 계층형 지식 메모, 음성 기록, 일정과 할 일을 한 화면에서 관리할 수 있습니다. 일상에서 떠오른 생각, 회의나 대화의 핵심, 여행과 살림 기록처럼 흩어지기 쉬운 정보를 NowNote에 모아두세요.

주요 기능
- 날짜별 메모: 하루 단위로 메모를 계속 추가하고 다시 확인
- 계층 메모: 부모/자식/손자 구조로 지식과 아이디어 정리
- 음성 기록: 말로 남긴 내용을 텍스트로 변환하여 저장
- 일정과 할 일: 오늘 일정, 루틴, 할 일을 홈에서 확인
- 로컬 중심 사용: 기본 기록은 기기 안에서 관리

NowNote는 입력을 줄이고 기록을 쉽게 만드는 것을 목표로 합니다. 복잡한 지식 관리 도구보다 가볍고, 단순한 메모장보다 연결된 개인 기록 환경을 제공합니다.
```

## 14. 업로드 키 생성

이 단계는 신산님 PC의 PowerShell에서 실행합니다. 비밀번호는 채팅에 남기지 않습니다.

작업 폴더 이동:

```powershell
cd D:\Project\Now\now_app\android
```

업로드 키 비밀번호를 현재 PowerShell 세션에만 설정합니다.

```powershell
$env:NOWNOTE_KEYSTORE_PASSWORD='직접정한_긴_비밀번호'
$env:NOWNOTE_KEY_PASSWORD='직접정한_긴_비밀번호'
```

업로드 키를 생성합니다.

```powershell
powershell -ExecutionPolicy Bypass -File .\create_upload_key.ps1
```

`keytool`을 자동으로 찾지 못하면 현재 PowerShell 세션에 아래 중 하나를 설정한 뒤 다시 실행합니다.

```powershell
$env:JAVA_HOME='JDK_설치_경로'
$env:NOWNOTE_KEYTOOL='keytool.exe_전체_경로'
```

생성되는 파일:

```text
D:\Project\Now\now_app\android\upload-keystore.jks
D:\Project\Now\now_app\android\key.properties
```

주의:

```text
upload-keystore.jks와 key.properties는 절대 Git에 올리면 안 됩니다.
업로드 키 비밀번호는 반드시 따로 안전하게 보관해야 합니다.
```

## 15. 서명된 AAB 빌드

업로드 키 생성 후 아래 명령을 실행합니다.

```powershell
powershell -ExecutionPolicy Bypass -File .\build_release_aab.ps1
```

Flutter 또는 JDK를 자동으로 찾지 못하면 아래 값을 현재 PowerShell 세션에 설정합니다.

```powershell
$env:NOWNOTE_FLUTTER_BIN='flutter.bat_전체_경로'
$env:NOWNOTE_JAVA_HOME='JDK_설치_경로'
```

성공하면 아래 파일이 생성됩니다.

```text
D:\Project\Now\now_app\build\app\outputs\bundle\release\app-release.aab
```

이 파일을 Play Console 내부 테스트 트랙에 업로드합니다.
