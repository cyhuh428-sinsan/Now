# NowNote 개인정보처리방침 초안

시행일: 2026년 5월 5일

공개 예정 URL: https://nownote.sinsan.kr/

NowNote는 사용자의 개인 기록을 안전하게 관리하기 위해 필요한 최소한의 권한과 데이터를 사용합니다.

## 수집 또는 처리하는 정보
NowNote는 앱 기능 제공을 위해 다음 정보를 기기 내에서 저장하거나 처리할 수 있습니다.

- 메모 및 기록 내용
- 음성 입력 및 음성 녹음 파일
- 사진 또는 이미지
- 일정 정보
- 할 일, 루틴, 생활 기록
- 사용자가 허용한 경우 Health Connect의 걸음, 운동, 수면, 활동 칼로리, 심박수 정보

## 이용 목적
처리되는 정보는 다음 목적에 사용됩니다.

- 메모 작성, 조회, 편집, 삭제
- 음성 기록의 텍스트 변환
- 일정, 할 일, 루틴 관리
- 기기 내 기록을 바탕으로 한 개인 기록 관리
- 사용자가 허용한 건강 데이터의 개인 생활 기록 보조

## 서버 전송 및 제3자 제공
기본 사용 방식에서는 기록이 사용자의 기기 안에 저장됩니다.

사용자가 NowNote 서버 연결을 켠 경우 메모, 녹음 파일, 텍스트 변환 결과, 분석 작업 입력 내용, 사용자 ID, 기기 ID, 동기화 시각이 사용자가 지정한 개인 서버 또는 공용 NowNote 서버로 전송될 수 있습니다.

서버 연결을 사용하지 않으면 기록은 기본적으로 기기 안에서 처리됩니다. 클라우드 동기화는 기본 제공하지 않습니다.

NowNote는 사용자의 개인 정보를 광고, 신용평가, 데이터 중개 목적으로 판매하거나 제공하지 않습니다.

## 권한 사용
NowNote는 기능 제공을 위해 다음 권한을 요청할 수 있습니다.

- 마이크: 음성 메모 및 음성 기록
- 카메라/사진: 캡처, 식사, 패션, 여행 등 생활 기록의 이미지 기록
- 일정: 일정 조회 및 관리
- 인터넷: 네트워크 연결 상태 확인 및 사용자가 켠 NowNote 서버 연결
- Health Connect: 사용자가 허용한 건강/활동 데이터 조회

## 보관 및 삭제
로컬에 저장된 데이터는 사용자가 앱에서 삭제하거나 앱 데이터를 삭제할 때 제거됩니다.

서버 연결 사용자는 앱의 삭제 기능과 서버 동기화를 통해 서버 기록 삭제 또는 비활성 백업 처리가 반영될 수 있습니다. 공용 서버에 저장된 데이터의 직접 삭제가 필요한 경우 운영자에게 삭제를 요청할 수 있습니다.

## 보안
NowNote는 사용자 데이터 보호를 위해 기기 권한, 로컬 저장소, 서버 API 토큰, 네트워크 전송 시 보안 절차를 적용합니다. 개인 서버 사용자는 API 토큰과 데이터베이스 비밀번호를 직접 안전하게 관리해야 합니다.
