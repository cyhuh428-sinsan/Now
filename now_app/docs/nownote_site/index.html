NowNote 개인정보처리방침

시행일: 2026년 5월 6일

공개 URL: https://nownote.sinsan.kr/

NowNote는 사용자의 개인 기록을 안전하게 관리하기 위해 필요한 최소한의 권한과 데이터를 사용합니다. 본 방침은 NowNote 앱에서 처리되는 개인정보와 기기 내 데이터의 이용 목적, 보관, 삭제, 권한 사용 방식을 설명합니다.

1. 처리하는 정보

NowNote는 앱 기능 제공을 위해 다음 정보를 기기 내에서 저장하거나 처리할 수 있습니다.

  • 메모 및 기록 내용
  • 음성 입력 및 음성 녹음 파일
  • 사진 또는 이미지
  • 일정 정보
  • 할 일, 루틴, 생활 기록
  • 사용자가 허용한 경우 Health Connect의 걸음, 운동, 수면, 활동 칼로리, 심박수 정보

2. 이용 목적

처리되는 정보는 다음 목적에만 사용됩니다.

  • 메모 작성, 조회, 편집, 삭제
  • 음성 기록의 저장 및 텍스트 변환
  • 일정, 할 일, 루틴 관리
  • 기기 내 기록을 바탕으로 한 개인 기록 관리
  • 사용자가 허용한 건강 데이터의 개인 생활 기록 보조

3. 서버 전송 및 제3자 제공

기본 사용 방식에서는 사용자의 기록이 기기 안에 저장됩니다. 현재 출시 버전은 LLM 분석, 서버 동기화, 클라우드 동기화 기능을 제공하지 않으며 사용자의 기록을 외부 서버로 전송하지 않습니다.

NowNote는 사용자의 개인정보를 광고, 신용평가, 데이터 중개 목적으로 판매하거나 제공하지 않습니다.

4. 권한 사용

NowNote는 기능 제공을 위해 다음 권한을 요청할 수 있습니다.

  • 마이크: 음성 메모 및 음성 기록
  • 카메라/사진: 이미지 기록 및 캡처
  • 일정: 일정 조회 및 관리
  • 알림: 일정, 루틴, 할 일 알림
  • 인터넷: 앱 기능 제공에 필요한 네트워크 연결 상태 확인
  • Health Connect: 사용자가 허용한 건강/활동 데이터 조회

5. 보관 및 삭제

로컬에 저장된 데이터는 사용자가 앱에서 삭제하거나 앱 데이터를 삭제할 때 제거됩니다. 현재 출시 버전은 서버 동기화 기능을 제공하지 않으므로 서버 저장 데이터 삭제 절차는 적용되지 않습니다.

6. 보안

NowNote는 사용자 데이터 보호를 위해 기기 권한, 로컬 저장소, 네트워크 전송 시 보안 절차를 적용합니다. 현재 출시 버전은 사용자의 기록을 외부 서버로 전송하지 않습니다.

7. 아동 개인정보

NowNote는 아동을 대상으로 개인정보를 의도적으로 수집하지 않습니다. 타깃 연령 및 대상 사용자는 Google Play Console 등록 정보에 따릅니다.

8. 방침 변경

본 개인정보처리방침은 앱 기능, 법령, Google Play 정책 변경에 따라 수정될 수 있습니다. 중요한 변경이 있는 경우 본 페이지를 통해 고지합니다.
