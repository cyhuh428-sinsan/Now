# NowNote 1.2 관계 탐색과 그래프뷰 설계서

이 문서는 NowNote 1.2 완료 여부를 판단하는 기준 문서입니다.

## 목표

NowNote 1.2의 목표는 Obsidian의 그래프뷰, 백링크, 나가는 링크, 연결 후보 기능 중 NowNote 지식 메모에 꼭 필요한 부분을 흡수하는 것입니다.

1.1에서 설치형 프로그램의 PC 로컬 원본과 Web의 서버 공유 문서 기준을 분리했으므로, 1.2에서는 각 클라이언트가 자기 기준의 문서 묶음을 관계 그래프로 볼 수 있어야 합니다.

## 제품 기준

| 클라이언트 | 그래프 기준 |
| --- | --- |
| Windows 설치형 | PC 로컬 원본 문서와 사용자가 공유로 선택한 문서 |
| Web | 로그인한 사용자의 서버 공유 문서 |
| 모바일 | 1.2 본구현 대상 아님. 이후 모바일 동등화 단계에서 경량 그래프 또는 관계 목록으로 확장 |
| 서버 | 그래프 계산 원본을 직접 저장하지 않고, 클라이언트 문서 관계를 보조하는 API/분석 후보로 유지 |

## 포함 범위

- 전체 그래프뷰
- 선택 메모 중심 로컬 그래프
- 로컬 그래프 깊이 1-3 선택
- 검색어 필터
- 태그 필터
- 그룹 기준: 주제, 태그, 공유 상태, 분석 상태
- 고립 메모 목록
- 허브 메모 목록
- 현재 메모의 outgoing links 목록
- 현재 메모의 backlinks 목록
- 연결 후보: 본문에 제목이 등장하지만 `[[제목]]`으로 연결되지 않은 메모
- 연결 후보를 `[[제목]]` 링크로 반영
- 그래프에서 메모 열기
- 그래프 필터 북마크 저장

## 제외 범위

- Canvas 보드
- 그래프 노드 수동 배치
- 서버 저장형 그래프 인덱스
- 임베딩 기반 의미 관계
- 모바일 전용 그래프 화면
- 실시간 공동 편집 관계 표시

## 화면 흐름

1. 사용자가 왼쪽 빠른 메뉴 또는 탐색 메뉴에서 `연결 보기`를 엽니다.
2. 기본은 전체 그래프입니다.
3. 선택 메모가 있으면 `선택 메모 중심`으로 바꿔 깊이 1-3의 로컬 그래프를 볼 수 있습니다.
4. 검색어, 태그, 그룹 기준을 바꿔 관계를 좁힙니다.
5. 아래 점검 패널에서 나가는 링크, 백링크, 연결 후보, 고립 메모, 허브 메모를 확인합니다.
6. 연결 후보를 누르면 현재 메모 본문에 `[[제목]]` 링크가 반영됩니다.
7. 자주 쓰는 필터 조합은 북마크로 저장하고 다시 적용합니다.

## 데이터 기준

- 그래프 노드는 삭제 보관함을 제외한 지식 메모입니다.
- 그래프 엣지는 메모 본문의 `[[메모 제목]]` 링크입니다.
- 암호화 메모는 잠금 상태에서 본문 관계를 추출하지 않습니다.
- Web은 서버 공유 문서만 로드하므로 그래프도 서버 공유 문서 기준입니다.
- 설치형은 Electron 로컬 저장소 기준으로 그래프를 만듭니다.
- 북마크는 설정 저장소에 보관하고 문서 데이터에는 섞지 않습니다.

## 영향 범위

- Web/설치형 공통 화면: 그래프 팝업 확장
- 설정 저장소: `settings.graph` 추가
- 기존 메모 저장 구조: 변경 없음
- 서버 API: 변경 없음
- 모바일 앱: 변경 없음

## 회귀 방지

- 기존 `[[링크]]`, 백링크, 태그 추출 기준을 유지합니다.
- 삭제 보관함은 그래프 대상에 포함하지 않습니다.
- 암호화 메모의 잠금 본문을 그래프 분석 대상으로 사용하지 않습니다.
- Web 공유 문서 전용 기준과 설치형 로컬 저장 기준을 바꾸지 않습니다.
