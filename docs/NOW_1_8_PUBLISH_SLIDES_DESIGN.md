# NowNote 1.8 출판, 발표, 공개 지식 묶음 설계서

## 목적

NowNote 1.8은 개인 지식 메모를 외부 결과물로 전환하는 단계입니다. Obsidian Publish와 Slides에서 필요한 부분만 흡수해, 사용자가 메모를 블로그형 공개 묶음과 발표 자료로 내보낼 수 있게 합니다.

## 범위

- 화면에서 공개 묶음 제목, 설명, permalink를 관리합니다.
- 공개 묶음에 포함할 지식 메모를 선택합니다.
- 공유 안 함, 암호화 메모, 공개 제외 속성, 비공개 태그는 자동 제외합니다.
- 공개 전 민감정보 후보를 화면에 표시합니다.
- 선택 묶음을 HTML 문서로 내보냅니다.
- 선택 묶음을 슬라이드형 HTML 문서로 내보냅니다.
- 화면에서 공개 묶음 미리보기를 확인합니다.

## 제외 범위

- 서버 호스팅형 공개 URL 발급
- PDF 렌더링 엔진 내장
- 첨부 이미지 파일 패키징
- 댓글, 구독, 검색 엔진 색인 설정
- Obsidian Publish 전체 호환

## 화면 기준

설정 화면에 `출판 / 발표` 영역을 둡니다.

- 묶음 선택
- 제목
- 설명
- permalink
- 포함 문서 체크 목록
- 미리보기
- 민감정보 점검 목록
- 공개 HTML 내보내기
- 발표 HTML 내보내기

## 공개 제외 기준

다음 문서는 외부 산출물에 포함하지 않습니다.

- `shared === false`
- 암호화된 본문
- 속성 `publicExclude`, `private`, `excludeFromPublish`가 true 계열 값
- 태그 `private`, `비공개`, `no-publish`

## 민감정보 점검 기준

다음 후보를 공개 전 경고로 표시합니다.

- 이메일 주소
- 전화번호 후보
- 비밀번호/토큰/API key 계열 단어
- 주민등록번호 형식 후보

## 데이터 기준

`state.data.publishBundles`에 공개 묶음 정보를 저장합니다.

```json
{
  "id": "bundle id",
  "title": "공개 묶음 제목",
  "description": "설명",
  "permalink": "public-path",
  "cover": "",
  "includeNodeIds": ["note id"],
  "createdAt": "ISO time",
  "updatedAt": "ISO time"
}
```

## 완료 기준

- 사용자가 화면에서 공개 묶음을 저장할 수 있습니다.
- 공개 제외 문서와 암호화 메모는 내보내기 결과에 포함되지 않습니다.
- 민감정보 후보가 내보내기 전 화면에 표시됩니다.
- 블로그형 HTML 내보내기 결과가 생성됩니다.
- 발표형 HTML 내보내기 결과가 생성됩니다.
- 1.8 설계서와 런타임 체크리스트 기준으로 완료 여부를 확인할 수 있습니다.
