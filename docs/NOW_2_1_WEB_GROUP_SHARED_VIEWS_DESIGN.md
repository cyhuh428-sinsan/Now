# NowNote 2.1 Web 그룹 공유 조회 설계서

기준일: 2026-06-01

## 문서 지위

- 이 문서는 NowNote 2.1 완료 여부를 확인하는 기준 문서다.
- 2.1은 2.0 LLM 지식화 이후, Web에서 그룹 공유 메모를 조회하는 화면을 분리하는 단계다.
- 그룹원 사이의 메신저는 2.2 범위로 분리한다.

## 목표

- Web에서 서버 공유 문서를 세 가지 기준으로 조회한다.
- 내 공유 메모는 조회와 수정이 가능해야 한다.
- 다른 그룹원의 공유 메모는 조회만 가능해야 한다.
- 설치형과 모바일 토큰 동기화에는 그룹 공유 조회 메뉴를 제공하지 않는다.

## 조회 메뉴

| 메뉴 | 목적 | 수정 가능 여부 |
| --- | --- | --- |
| 내 공유메모 | 내가 서버에 공유한 지식 메모를 조회하고 수정한다. | 가능 |
| 그룹 지식체계 | 같은 그룹의 공유 메모를 전체 지식체계 기준으로 조회한다. | 내 메모만 가능, 그룹원 메모는 불가 |
| 구성원별 공유문서 | 그룹 구성원별로 공유 문서를 묶어서 조회한다. | 내 메모만 가능, 그룹원 메모는 불가 |

## 화면 기준

- 세 메뉴는 hosted Web에서만 표시한다.
- 설치형, 모바일, 로컬 파일 실행 화면에는 메뉴를 표시하지 않는다.
- Web 로그인 후 서버 공유 문서를 불러올 때 같은 그룹원의 공유 메모를 읽기 전용으로 포함한다.
- 그룹원 메모는 `groupSharedReadOnly` 상태로 표시한다.
- 그룹원 메모는 제목, 본문, 속성, 공유 상태, 삭제, 이동, 암호화 편집이 비활성화된다.

## 데이터 기준

- 내 공유 메모는 기존 로컬 ID를 유지한다.
- 그룹원 공유 메모는 `group:{owner_id}:{local_id}` 형식의 Web 내부 ID를 사용한다.
- 그룹원 공유 메모의 원본 소유자는 `remoteOwnerId`로 표시한다.
- 그룹원 공유 메모는 서버 업로드 대상에서 제외한다.
- 그룹원 공유 메모는 삭제 보관함이나 로컬 변경 대기 목록에 섞지 않는다.

## 완료 기준

- Web에 세 조회 메뉴가 표시된다.
- 내 공유메모 메뉴는 내 공유 메모만 표시한다.
- 그룹 지식체계 메뉴는 내 공유 메모와 같은 그룹원의 공유 메모를 함께 표시한다.
- 구성원별 공유문서 메뉴는 소유자별 섹션으로 공유 메모를 표시한다.
- 그룹원 공유 메모는 읽기 전용으로 열리고 수정 버튼이 비활성화된다.
- 설치형/로컬 화면에서는 세 조회 메뉴가 숨겨진다.
- 검증 스크립트가 세 메뉴와 읽기 전용/업로드 제외 기준을 확인한다.
