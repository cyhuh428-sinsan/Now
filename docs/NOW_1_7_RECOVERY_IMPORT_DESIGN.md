# NowNote 1.7 복구, 가져오기, 마이그레이션 설계서

기준일: 2026-06-01

## 목적

NowNote 1.7의 목표는 사용자가 데이터 손상, 실수, 외부 도구 이전을 화면에서 처리할 수 있게 하는 것입니다.
Obsidian의 File recovery, Format converter, Importer 성격을 NowNote 운영형 기능으로 흡수합니다.

## 포함 범위

- 화면 기반 로컬 스냅샷
- JSON/Markdown 가져오기 전 자동 스냅샷
- 선택 스냅샷 복구
- Markdown/Obsidian 가져오기 보정
- 기본 frontmatter와 NowNote 속성 매핑
- Obsidian wiki 링크와 첨부 표기 보정
- 가져오기 후 문제/보정 목록 표시
- 중복 제목 후보 표시

## 제외 범위

- 서버 전체 백업 복구
- 대용량 첨부 원본 복구
- Obsidian 플러그인별 특수 문법 완전 변환
- JSON Canvas 전체 호환 가져오기
- 자동 병합 결정

## 데이터 기준

1.7은 `state.data.snapshots`와 `state.data.importReports`를 사용합니다.
스냅샷은 최근 12개까지만 보관하며, 대형 첨부 원본은 1.7 범위에서 복구 대상으로 보지 않습니다.

## 안전 기준

- 복구는 사용자의 명시적 확인 후 실행합니다.
- 가져오기 전에는 자동 스냅샷을 먼저 남깁니다.
- 스냅샷 복구 시에도 현재 상태를 다시 스냅샷으로 남깁니다.
- frontmatter는 NowNote 속성으로 매핑하되 원문 본문에는 노출하지 않습니다.
- 변환하지 못한 항목은 삭제하지 않고 진단 목록에 남깁니다.
