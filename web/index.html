서버 연결

단독 사용 또는 개인/공용 NowNote 서버 연결 방식을 선택합니다.

입력 기준
  • 단독 사용: 서버 없이 이 기기에만 저장합니다.
  • 설치형/앱: 서버 주소, 사용자 ID, 사용자별 접속 토큰을 입력합니다.
  • Web: 서버 주소에서 사용자 ID와 비밀번호로 로그인합니다.
  • 앱/설치형 접속 토큰은 Web 로그인 후 내 연결 토큰 화면에서 직접 발급합니다.
사용자 프로필

프로필을 불러오지 않았습니다.

앱/설치형 연결 토큰

앱이나 설치형 프로그램을 공용 서버에 연결할 때 여기서 토큰을 직접 발급합니다. 발급된 값은 한 번만 보입니다.
